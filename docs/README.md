# Docs (GitHub Pages)

Contenido público y enmascarado de Operación Meridiano.

- `index.html`: War Room público.
- `styles.css`: estilos extraídos.
- `app.js`: lógica interactiva extraída.
- `data_public.js`: datos sin sensibilidad (sin rentabilidad en Bs).

## Actualización de Pages

1. Actualizar `docs/data_public.js` con versión pública.
2. Verificar que `docs/` no incluya XLSX ni CSV sensibles.
3. Publicar desde branch `main`, carpeta `/docs`.
